Store Monitoring System API

To hit the trigger_report API, follow the below process:

  • Step 1: Open Postman or any API testing applications
  • Step 2: Make a POST request
  • Step 3: Hit the URL - localhost:5000/api/trigger_report
  • Step 4: Copy the random string

To hit the get_report API, follow the below process:

  • Step 1: Open any browser.
  • Step 2: Hit the URL - localhost:5000/api/get_report?report_id=random_string
  • Step 3: CSV file will generate
